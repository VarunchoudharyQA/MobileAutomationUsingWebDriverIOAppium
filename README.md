# Webdriver.io-Appium-Cucumber-Framewrok
> Implemented POM and Cucumber Setup
> 

## Tools Required:
- npm or yarn
- Andriod Studio

## Setup:
- run `npn i` to setup all the required packages from `package.json`


**To run the code use:**
- `npm run wdio`
      
**To Generate Allure Reports:**
- `allure-generate`
- `allure-open`
